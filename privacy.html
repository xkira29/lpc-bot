Política de Privacidad

LPC Bot · Liga Profesional de Creadores

Última actualización: julio de 2026

1. Información que puede recopilar el bot

LPC Bot puede procesar información básica necesaria para funcionar dentro de Discord, incluyendo:

  • Identificadores de usuarios de Discord.
  • Identificadores de servidores.
  • Identificadores de canales.
  • Información relacionada con mejoras o boosts del servidor.

2. Uso de la información

La información se utiliza exclusivamente para proporcionar las funciones del bot, como enviar avisos, registrar eventos y mostrar mensajes relacionados con el servidor.

3. Compartición de datos

No vendemos, alquilamos ni compartimos información personal con terceros, excepto cuando sea necesario para cumplir con la ley.

4. Almacenamiento de datos

El bot solo conservará la información estrictamente necesaria para su funcionamiento. Algunos datos pueden no almacenarse de forma permanente.

5. Eliminación de datos

Los usuarios o administradores pueden solicitar información sobre sus datos o pedir su eliminación mediante el correo de contacto.

6. Contacto

Para dudas relacionadas con esta política, puedes escribir a: galacticstudios256@gmail.com
